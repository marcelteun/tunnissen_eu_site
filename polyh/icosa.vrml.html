Introduction

In the book The 59 Icosahedra H.S.M. Coxeter presents all possible stellations of the icosahedron according to a set of rules. This page presents these as 3D models so that the reader can investigate them further.

To be able to view and investigate the 3D models you need a VRML player. See note2.
