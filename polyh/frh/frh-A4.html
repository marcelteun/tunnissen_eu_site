Fold Position 1

3D Model Extra Info Link
This one is quite interesting. First of all there are four seperate vertices on 3-fold axes that all end up in the same point: the centre of the polyhedron. Another thing is that parts of different heptagons and up in the same plane, but these flaps aren't loose: the can only be in this one position. If that isn't special enough: it is the only possibility found for a W-fold in this position.
