Introduction

This page summarises which polyhedra were found that only consist of regular folded heptagons with A4 x I symmetry. Look here for general background information.The listing is sub-divided into the different heptagon folds.

Note: for optimal experience Javascript should be enabled.

Here two different faces fall into one plane and form flaps. More variants exist since one can flip the flaps inwards and because there are different kinds, different combinations exist.
This is similar to the previous one, but the triangles are folded inwards. Similarly to the previous ones there are variations with different flaps.

Heptagons folded using a W Fold

3D Model Extra Info Link
Nice and simple model. This was one of the first models I built. For this one I proved that it works for regular heptagons, not just numerically.
Quite a neat one, but difficult to build with the deep small cracks.
This is a funny one: here two parts are folded 180 degrees so that 3 pieces end flat on top of each other.
On first sight it looks rather simple, but it has deep small cracks, so deep that they stick out into the crack at the opposite side. These cracks have some complicating inside. If you check inside you would see how the different heptagons connect with each other.
This one has a lot going on. Neighbouring heptagons interscect each other so that two volumes only connect through an edge. It also have some beautiful cavings near the 2-fold axes. It looks all complicated, but it is still easy to recognise the folded heptagons.

Heptagons folded using a Shell Fold

3D Model Extra Info Link
Another polyhedron that is one of the first 'follyheptahedra' I built. It looks as if it consists of three 'H', which could be interpreted as Heptagons in 3D. Here you can see that with a shell-fold it is possible to glue together two neighbouring edges to obtain sort of a cone. For this one I proved that it really works with regular helptagons and that it isn't just a near miss.
This looks a bit like the previous one, but with many more self-intersections.
This is one I like. It is a nice star shape, but the star points seems to have turned inwards so far that they still stick out. The star points consists of the cones that I mentioned above.
This one looks a bit like the previous one, but the cones consists of self-intersecting heptagons.
This one also consists of self-intersecting heptagons. When I generated this first I filtered this one out as 'not so interesting', but now at a second glance, it is actually quite nice.
This one is similar to the previous one, but instead having self-intersecting heptagons, the parts that self-intersect are pushed inwards and as a consequence the star points don't intersect either. Now they remind me a bit of Han Solo's Millennium Falcon.
